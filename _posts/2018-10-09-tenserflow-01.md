## 텐서플로우 설치하기 

파이선 설치:
``` 
https://www.python.org/downloads/
```

텐서플로우 설치:
```
C:\> pip3 install --upgrade tensorflow
```

라이브러리 설치:
```
C:\> pip3 install numpy matplotlib pillow
```

주피터 노트북 설치:
```
C:\> pip3 install jupyter
```

주피터 노트북 실행:
```
C:\> jupyter notebook
```
